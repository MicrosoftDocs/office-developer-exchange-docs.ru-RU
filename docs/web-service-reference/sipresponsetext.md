# <a name="sipresponsetext"></a>SIPResponseText

Элемент **SIPResponseText** указывает текст ответа SIP. 
  
```xml
<SIPResponseText/>
```

 **string**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[PhoneCallInformation](phonecallinformation.md) <br/> |Задает сведения о состоянии для телефонного звонка.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текст ответа SIP.
  
## <a name="remarks"></a>Замечания

Схема, описывающая этот элемент находится в каталоге /ews/ компьютера, на котором работает Exchange Server с установленной ролью сервера клиентского доступа.
