# <a name="suppressreadreceipt"></a>суппрессреадрецеипт

Элемент **суппрессреадрецеипт** используется для подавления уведомлений о прочтении. 
  
```xml
<SuppressReadReceipt>
   <ReferenceItemId/>
</SuppressReadReceipt>
```

 **суппрессреадрецеипттипе**
