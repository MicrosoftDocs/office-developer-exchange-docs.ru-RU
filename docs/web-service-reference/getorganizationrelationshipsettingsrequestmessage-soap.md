# <a name="getorganizationrelationshipsettingsrequestmessage-soap"></a>GetOrganizationRelationshipSettingsRequestMessage (SOAP)

Элемент **GetOrganizationRelationshipSettingRequestMessage** представляет запрос операции [GetOrganizationRelationshipSettings операции (SOAP)](getorganizationrelationshipsettings-operation-soap.md) . Элемент **GetOrganizationRelationshipSettingRequestMessage** — только для внутреннего использования. Этот элемент не используется с клиентами. 
  
```XML
<GetOrganizationRelationshipSettingRequestMessage>
   <Request/>
</GetOrganizationRelationshipSettingRequestMessage>
```

 **GetOrganizationRelationshipSettingsRequest**
