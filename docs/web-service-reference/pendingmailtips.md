# <a name="pendingmailtips"></a>пендингмаилтипс

Элемент **пендингмаилтипс** указывает на то, что советы по использованию электронной почты в этом элементе невозможно оценить до истечения времени ожидания обработки на сервере. 
  
```XML
<PendingMailTips>All | OutOfOfficeMessage | MailboxFullStatus | CustomMailTip | ExternalMemberCount | TotalMemberCount | MaxMessageSize | DeliveryRestriction | ModerateStatus | InvalidRecipient</PendingMailTips>
```

 **маилтиптипес**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Подсказки](mailtips.md) <br/> |Представляет значения для различных типов советов по использованию электронной почты.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

В следующей таблице приведены возможные значения для элемента **пендингмаилтипс** . 
  
|**Значение**|**Описание**|
|:-----|:-----|
|Все  <br/> |Представляет все доступные советы по использованию электронной почты.  <br/> |
|аутофоффицемессаже  <br/> |Представляет сообщение об отсутствии на рабочем месте (отсутствие на работе).  <br/> |
|маилбоксфуллстатус  <br/> |Представляет состояние для почтового ящика Full.  <br/> |
|кустоммаилтип  <br/> |Представляет настраиваемую подсказку почты.  <br/> |
|екстерналмемберкаунт  <br/> |Представляет количество внешних элементов.  <br/> |
|тоталмемберкаунт  <br/> |Представляет количество всех элементов.  <br/> |
|MaxMessageSize  <br/> |Представляет максимальный размер сообщения, которое может принимать получатель.  <br/> |
|деливерирестриктион  <br/> |Указывает, будут ли ограничения доставки допустить достижение получателю сообщения отправителя.  <br/> |
|модератионстатус  <br/> |Указывает, будет ли сообщение отправителя проверено модератором.  <br/> |
|инвалидреЦипиент  <br/> |Указывает, является ли получатель недопустимым.  <br/> |
   
## <a name="remarks"></a>Примечания

Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
