# <a name="request-getfederationinformation-soap"></a>Запрос (SOAP) (GetFederationInformation)

Элемент **запроса** представляет запрос [GetFederationInformationRequest (SOAP)](getfederationinformationrequest-soap.md) . 
  
```XML
<Request>
   <Domain/>
</Request>
```

 **GetFederationInformationRequest**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Домен (GetFederationInformation) (SOAP)](domain-getfederationinformationsoap.md) <br/> |Определяет домен, имеющей отношение доверия федерации.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[GetFederationInformationRequestMessage (SOAP)](getfederationinformationrequestmessage-soap.md) <br/> |Подготавливает звонка на сервере для запроса данных конфигурации для службы маркеров безопасности (STS).  <br/> |
