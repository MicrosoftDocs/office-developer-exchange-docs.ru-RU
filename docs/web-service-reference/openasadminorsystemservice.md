# <a name="openasadminorsystemservice"></a>опенасадминорсистемсервице

Элемент **опенасадминорсистемсервице** предназначен только для внутреннего использования. Этот элемент не используется клиентами. 
  
```XML
<OpenAsAdminOrSystemService LogonType="">
   <ConnectingSID/>
</OpenAsAdminOrSystemService>
```

 **опенасадминорсистемсервицетипе**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

|**Атрибут**|**Описание**|
|:-----|:-----|
|LogonType  <br/> |Не предназначено для использования клиентом.  <br/> |
   
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|коннектингсид  <br/> |Не предназначено для использования клиентом.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

Отсутствуют.
  
## <a name="text-value"></a>Текстовое значение

Нет.
  
## <a name="remarks"></a>Примечания

Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.Этот элемент появился в Exchange Server 2010 с пакетом обновления 1 (SP1).
