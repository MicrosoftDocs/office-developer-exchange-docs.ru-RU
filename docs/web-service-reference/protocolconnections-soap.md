# <a name="protocolconnections-soap"></a>ProtocolConnections (SOAP)

Элемент **ProtocolConnections** содержит ноль или больше протокол подключения. 
  
```XML
<ProtocolConnections>
   <ProtocolConnection/>
</ProtocolConnections>
```

 **ProtocolConnections**
