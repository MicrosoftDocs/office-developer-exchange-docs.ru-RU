# <a name="cchksgfileserrgetheader-function-reserved"></a>Функция функция cchksgfiles. Ерржесеадер (зарезервированная)

**Применимо к:** Exchange Server 2003 | Exchange Server 2007 | Exchange Server 2010 | Exchange Server 2013
  
Зарезервировано для будущего использования и не реализовано. Не вызывайте эту функцию. 
  
```cs
Vitual ERRErrGetHeader  
(
    __in_z const WCHAR  * const wszFile,
        VOID  * const pvResult,
    Const ULONGcbMax,
    Const ULONGulFlags = NO_FLAGS
);

```

## <a name="parameters"></a>Параметры

### <a name="wszfile"></a>всзфиле
  
Зарезервировано для будущего использования; не используйте.
    
### <a name="pvresult"></a>пвресулт
  
Зарезервировано для будущего использования; не используйте.
    
### <a name="cbmax"></a>кбмакс
  
Зарезервировано для будущего использования; не используйте.
    
### <a name="ulflags"></a>ulFlags
  
Зарезервировано для будущего использования; не используйте.
    
## <a name="return-value"></a>Возвращаемое значение

Нет.
  
## <a name="remarks"></a>Примечания

Нет.
  
## <a name="requirements"></a>Requirements

Exchange Server 2013 включает в себя только 64-разрядную версию API ЧКСГФИЛЕС.
