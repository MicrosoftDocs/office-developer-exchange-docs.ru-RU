# <a name="minorversion-soap"></a>MinorVersion (SOAP)

Элемент **MinorVersion** представляет дополнительный номер версии для сервера. 
  
```XML
<MinorVersion/>
```

 **int**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[ServerVersionInfo (SOAP)](serverversioninfo-soap.md) <br/> |Содержит версию сервера, на обработку запроса.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Целочисленное значение, представляющее дополнительный номер версии сервера, на котором обработать запрос.
