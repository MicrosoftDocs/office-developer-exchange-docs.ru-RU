# <a name="contains"></a>Хранится

Элемент **Contains** представляет выражение поиска, которое определяет, содержит ли данное свойство предоставленное значение константы String. 
  
```xml
<Contains ContainmentMode="" ContainmentComparison="">
   <FieldURI/>
   <Constant/>
</Contains>
```

```xml
<Contains ContainmentMode="" ContainmentComparison="">
   <ExtendedFieldURI/>
   <Constant/>
</Contains>
```

```xml
<Contains ContainmentMode="" ContainmentComparison="">
   <IndexedFieldURI/>
   <Constant/>
</Contains>
```


**контаинсекспрессионтипе**

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

|**Атрибут**|**Описание**|
|:-----|:-----|
|**контаинментмоде** <br/> |Определяет границы поиска.  <br/> |
|**контаинменткомпарисон** <br/> |Определяет, будут ли при поиске игнорироваться регистры и пробелы.  <br/> |
   
#### <a name="containmentmode-attribute-values"></a>Значения атрибутов Контаинментмоде

|**Значение**|**Описание**|
|:-----|:-----|
|фуллстринг  <br/> |Между полной строкой и константой выполняется сравнение. Значение свойства и заданная константа в точности совпадают.  <br/> |
|Префиксом  <br/> |Сравнение между префиксом строки и константой.  <br/> |
|Подстроку  <br/> |Сравнение между подстрокой строки и константой.  <br/> |
|префиксонвордс  <br/> |В качестве префикса для отдельных слов в строке и константы используется сравнение.  <br/> |
|ексактфрасе  <br/> |Точная фраза в строке и константа сравнивается.  <br/> |

## <a name="remarks"></a>Примечания

Атрибуты используются для определения способа сравнения элементов.
  
Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере с MicrosoftExchange Server 2007 и установленной ролью сервера клиентского доступа.
