Информация, содержащаяся в этом документе, может относиться к функциям и продуктам предварительной версии и может претерпеть значительные изменения до окончательного коммерческого выпуска. Настоящий документ предоставляется "как есть" и служит только для информационных целей. Корпорация Майкрософт не предоставляет никаких гарантий, явных или подразумеваемых, в связи с этим документом Элемент **RedirectToRecipients** указывает адреса электронной почты, к которым должны перенаправляться сообщения. 
  
```XML
<RedirectToRecipients>
   <Address/>
</RedirectToRecipients>
```
