# <a name="categories"></a>Категории

Элемент **Categories** содержит коллекцию строк, определяющих категории, к которым относится элемент в почтовом ящике. 
  
```XML
<Categories>
   <String/>
</Categories>
```

 **аррайофстрингстипе**
