# <a name="imcontactsipuriaddress"></a>имконтактсипуриаддресс

Элемент **имконтактсипуриаддресс** содержит URI SIP-адрес контакта, добавляемого в группу обмена мгновенными сообщениями (IM). 
  
```XML
<ImContactSipUriAddress></ImContactSipUriAddress>
```

 **нонемптистрингтипе**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.
  
### <a name="child-elements"></a>Дочерние элементы

Отсутствуют.
  
### <a name="parent-elements"></a>Родительские элементы

[AddNewTelUriContactToGroup](addnewteluricontacttogroup.md)
  
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **имконтактсипуриаддресс** — это URI SIP-адрес контакта. 
  
## <a name="remarks"></a>Примечания

Этот элемент появился в Exchange Server 2013.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
