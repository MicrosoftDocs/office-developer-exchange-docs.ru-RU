# <a name="fileextension-soap"></a>Добавляемое (SOAP)

Элемент **добавляемое** представляет расширение файла. 
  
```XML
<FileExtension/>
```

 **string**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[SupportedFileExtensions (SOAP)](supportedfileextensions-soap.md) <br/> |Содержит список расширений файлов в документе общего ресурса.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **добавляемое** является допустимое расширение файла для документа, общий доступ к расположения. 
  
## <a name="element-information"></a>Сведения об элементе

|||
|:-----|:-----|
|Пространство имен  <br/> |http://schemas.microsoft.com/exchange/2010/Autodiscover  <br/> |
|Имя схемы  <br/> |Схема службы автообнаружения  <br/> |
|Файл проверки  <br/> |Messages.xsd  <br/> |
|Может быть пустым  <br/> |True  <br/> |
