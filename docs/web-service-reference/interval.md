# <a name="interval"></a>Интервал

Элемент **Interval** определяет интервал между двумя последовательными повторяющимися элементами. 
  
```xml
<Interval/>
```

 **int**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.
  
### <a name="child-elements"></a>Дочерние элементы

Отсутствуют.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[абсолутемонслирекурренце](absolutemonthlyrecurrence.md) <br/> |Представляет ежемесячный шаблон повторения.  <br/> |
|[даилиреженератион](dailyregeneration.md) <br/> |Описывает частоту повторного создания задачи (в днях).  <br/> |
|[даилирекурренце](dailyrecurrence.md) <br/> |Описывает частоту повторения задачи в днях.  <br/> |
|[монслиреженератион](monthlyregeneration.md) <br/> |Описывает частоту повторного создания задачи в месяцах.  <br/> |
|[релативемонслирекурренце](relativemonthlyrecurrence.md) <br/> |Описывает относительный месячный шаблон для повторяющейся задачи.  <br/> |
|[виклирекурренце](weeklyrecurrence.md) <br/> |Описывает частоту в неделях (в неделях), в течение которых задача повторяется.  <br/> |
|[виклиреженератион](weeklyregeneration.md) <br/> |Описывает частоту создания задачи в неделях.  <br/> |
|[еарлиреженератион](yearlyregeneration.md) <br/> |Описывает частоту создания задачи в годах.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение, представляющее целое число, обязательно.
  
## <a name="remarks"></a>Примечания

Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере с MicrosoftExchange Server 2007 и установленной ролью сервера клиентского доступа.
