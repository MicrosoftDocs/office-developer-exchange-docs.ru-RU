# <a name="action-protectionruleactiontype"></a>Действие (ProtectionRuleActionType)

Элемент **Action** определяет, какое действие необходимо выполнить Если соответствует части условие правила. 
  
```xml
<Action Name="">
   <Argument/>
</Action>

```

 **ProtectionRuleActionType**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

|**Атрибут**|**Описание**|
|:-----|:-----|
|**Name** <br/> |Определяет имя действия.  <br/> |
   
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Аргумент](argument.md) <br/> |Задает аргументов, то действие. Этот элемент не возникает, если заданное действие не требует аргументов. Этот элемент может возникнуть один или несколько раз, если действие, которое требуется для одного или нескольких аргументов. Действие **RightsProtectMessage** будет содержать один аргумент.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Rule](rule.md) <br/> |Содержит правило одного защиты.  <br/> |
