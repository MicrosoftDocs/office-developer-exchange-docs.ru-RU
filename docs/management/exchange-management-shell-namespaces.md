# <a name="exchange-management-shell-namespaces"></a>Пространства имен командной консоли Exchange

Найдите сведения о пространствах имен для командной консоли Exchange в Exchange.
  
**Применимо к:** Exchange Online | Exchange Server 2013 | Office 365
  
Приложения командной консоли Exchange для Exchange Online, Exchange Online в составе Office 365 или версия Exchange, начинающаяся с Exchange 2013, взаимодействуют с средой Windows PowerShell с помощью типов, предоставляемых пространством имен **System. Management. Automation** . Командлеты командной консоли Exchange используют типы из нескольких библиотек Exchange. В этом разделе представлены справочные сведения о типах и интерфейсах из библиотек Exchange, которые используются командлетами командной консоли Exchange.
