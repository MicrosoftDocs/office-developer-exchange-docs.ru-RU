# <a name="encryptionmethod-soap"></a>EncryptionMethod (SOAP)

Элемент **EncryptionMethod** представляет криптографический метод, используемый для протоколов POP, IMAP и SMTP. 
  
```XML
<EncryptionMethod/>
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Протоколконнектион (SOAP)](protocolconnection-soap.md) <br/> |Представляет подключение протокола для веб-клиента сервера.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение этого элемента — это криптографический метод, используемый для протоколов POP, IMAP и SMTP.
