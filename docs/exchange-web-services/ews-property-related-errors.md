# <a name="ews-property-related-errors"></a>Ошибки, связанные с свойство веб-служб Exchange

Узнайте, как обрабатывать ошибки, связанные с свойство в приложении веб-служб Exchange.
  
Большинство клиентских приложений веб-служб Exchange будет использовать свойства, что означает, что необходимо обрабатывать ошибки, связанные с свойство. Может обрабатывать эти ошибки во время выполнения или при разработке приложения веб-служб Exchange.
  
**В таблице 1: Ошибки, связанные с свойство и порядок их обработки**

|**Error**|**Возникающие при попытке...**|**Обработать его с...**|
|:-----|:-----|:-----|
|ErrorDataSizeLimitExceeded  <br/> |Задайте для свойства значение, превышающее максимальный размер для свойства или свойства не поддерживает потоковая передача по технологии, такие как свойства папки.  <br/> |Ограничение размера данных, задайте для свойства.  <br/> |
|ErrorFolderPropertRequestFailed  <br/> |Получите свойство, которое не удалось получить.  <br/> |Указывает, что свойство не удается получить.  <br/> |
|ErrorInvalidExtendedProperty  <br/> |Задайте Недопустимое сочетание значений расширенных свойств или результаты в недопустимом расширенные свойства универсальный код ресурса (URI).  <br/> |Проверка значения расширенные свойства.  <br/> |
|ErrorInvalidExtendedPropertyValue  <br/> |Задайте значение расширенного свойства, которое не соответствует указанному типу  <br/> |Обновление кода для проверки соответствия типов.  <br/> |
|ErrorInvalidFolderId  <br/> |Значение структуры идентификатор папки недопустимый формы.  <br/> |Только с использованием идентификаторов возвращаемых веб-служб Exchange.  <br/> |
|ErrorInvalidId  <br/> |Задайте структура идентификатора и/или изменить ключ на недопустимое формы.  <br/> |Только с использованием идентификаторов возвращаемых веб-служб Exchange.  <br/> |
|ErrorInvalidIdEmpty  <br/> |Задайте пустой идентификатор.  <br/> |Настройка идентификатора с допустимым идентификатором элемента или папки.  <br/> |
|ErrorInvalidIdMalformed  <br/> |Задайте структура идентификатора и/или изменить ключ на недопустимое формы.  <br/> |Только с использованием идентификаторов возвращаемых веб-служб Exchange.  <br/> |
|ErrorInvalidPropertyAppend  <br/> |Добавьте свойство, которое не поддерживает добавление.  <br/> |Обновление кода, чтобы она пытается добавить значения свойства набор получателей (Кому, копия, Bcc), свойства коллекции Attendee (требуется, необязательно, ресурсы), свойство Body и свойство ReplyTo.  <br/> |
|ErrorInvalidPropertyDelete  <br/> |Удаление свойства, которое не поддерживает удаление.  <br/> |Обновление кода, не пытайтесь удалить свойство. Например невозможно удалить папку и идентификаторы элементов.  <br/> |
|ErrorInvalidPropertyForExists  <br/> |Задайте ограничения существования на основе поиска на основе флаг свойство.  <br/> |Обновление кода, не используйте свойства на основе флаг в ограничения существования на основе поиска. На основе флаг свойства — это IsDraft, IsSubmitted, IsUnmodified, IsResend и IsFromMe.  <br/> |
|ErrorInvalidPropertyForOperation  <br/> |Работать с свойства элемента или папки, не поддерживаемые операции.  <br/> |Обновление кода, не доступа к свойству с помощью операции, вызвавшей ошибку.  <br/> |
|ErrorInvalidPropertyRequest  <br/> |Задает свойство в запросе, который не поддерживается для типа элемента.  <br/> |Обновление кода, не пытается получить доступ к свойству с помощью операции.  <br/> |
|ErrorInvalidPropertySet  <br/> |Необходимо задайте свойство только для чтения.  <br/> |Обновление кода, не пытайтесь задать свойства.  <br/> |
|ErrorInvalidValueForProperty  <br/> |Сравните значение свойства в качестве ограничения поиска, где значение сравнения не соответствует тип свойства.  <br/> |Обновление кода на наличие Несоответствие типа свойства.  <br/> |
|ErrorItemSavePropertyError  <br/> |Сохраните значения недопустимые свойства элемента или папки.  <br/> |Проверка значений свойств и типы перед отправкой их в запросе.  <br/> |
|ErrorNoFolderClassOverride  <br/> |Задайте класса папки на новую папку, которая не является типом основной папки.  <br/> |Задайте класса папки с помощью типа общей папки.  <br/> |
|ErrorNoPropertyTagForCustomProperties  <br/> |Ссылаться на дополнительное пользовательское свойство с его свойство тег.  <br/> |Обновление кода ссылок на настраиваемые дополнительный идентификатор набора свойств, свойств и имя свойства или идентификатор свойства.  <br/> |
|ErrorObjectTypeChanged  <br/> |Установить или обновить класс элемента на элемент, который не совпадает с типом схемы.  <br/> |Обновление кода, чтобы класс элемента совпадает с типом элемента схемы.  <br/> |
|ErrorPropertyUpdate  <br/> |Обновите свойство с недопустимое значение свойства.  <br/> |Проверка значения свойства перед отправкой в запросе [UpdateItem](http://msdn.microsoft.com/library/5d027523-e0bc-4da2-b60b-0cb9fc1fdfe4%28Office.15%29.aspx) .  <br/> |
|ErrorRequiredPropertyMissing  <br/> |Отправьте запрос CreateAttachment, где отсутствуют необходимые свойства.  <br/> |Обновление кода, присвойте свойству отсутствующие в соответствии с путь свойства, возвращаемого в ответе.  <br/> |
|ErrorUnsupportedMapiPropertyType  <br/> |Используйте типы расширенных свойств тип объекта, массив объектов, ошибки или null.  <br/> |Обновление кода, не используйте типы ограниченных расширенных свойств.  <br/> |
|ErrorUnsupportedPathForQuery  <br/> |Используйте путь неподдерживаемые свойства в качестве ограничения поиска.  <br/> |Изменение ограничения поиска, чтобы исключить путь неподдерживаемые свойства.  <br/> |
|ErrorUnsupportedPathForSortGroup  <br/> |Используйте путь неподдерживаемые свойства в отсортированный или сгруппированные поисковый запрос.  <br/> |Изменение ограничения поиска, чтобы исключить путь неподдерживаемые свойства.  <br/> |
|ErrorUnsupportedTypeForConversion  <br/> |Тип свойства, которое не может быть преобразована XML для веб-служб Exchange для возврата в ответ на запрос.  <br/> |Обновление кода, не запрашивать свойство не поддерживается.  <br/> |
|ErrorUpdatePropertyMismatch  <br/> |Обновление элемента или папки Описание изменений, для которого не совпадает с свойство, заданный для обновления.  <br/> |Изменение кода, чтобы изменить описание совпадает с типом элемента или папки, которая обновляется.  <br/> |
