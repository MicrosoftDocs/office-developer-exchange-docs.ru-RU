# <a name="mailtipsaccessenabled-soap"></a>MailTipsAccessEnabled (SOAP)

Элемент **MailTipsAccessEnabled** представляет флаг **MailTipsAccessEnabled()** . Элемент **MailTipsAccessEnabled** — только для внутреннего использования. Этот элемент не используется с клиентами. 
  
```XML
<MailTipsAccessEnabled>true | false</ MailTipsAccessEnabled>
```

 **Boolean**
