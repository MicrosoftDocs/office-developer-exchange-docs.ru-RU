# <a name="applicationuri-soap"></a>ApplicationUri (SOAP)

Элемент **ApplicationUri** определяет расположение приложения. 
  
```XML
<ApplicationUri/>
```

 **anyURI**

### <a name="child-elements"></a>Дочерние элементы
