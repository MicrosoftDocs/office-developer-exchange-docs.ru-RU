# <a name="ruleoperationerror"></a>RuleOperationError

Элемент **RuleOperationError** — ошибка при операции правила. 
  
```XML
<RuleOperationError>
    <OperationIndex/>
    <ValidationErrors/>
</RuleOperationError>
```

 **RuleOperationErrorType**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[OperationIndex](operationindex.md) <br/> |Указывает индекс операции в запросе, вызвавшей ошибку операции правила.  <br/> |
|[ValidationErrors](validationerrors.md) <br/> |Представляет массив ошибок проверки правил для каждого правила поля, имеющего ошибку.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[RuleOperationErrors](ruleoperationerrors.md) <br/> |Представляет массив ошибок проверки правил для каждого правила поля, имеющего ошибку.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Нет.
  
## <a name="remarks"></a>Замечания

Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
