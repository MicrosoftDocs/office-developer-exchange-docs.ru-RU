# <a name="numberofmemberswithconflict"></a>NumberOfMembersWithConflict

Элемент **NumberOfMembersWithConflict** представляет число членов списка рассылки, конфликт с временем предложенного собрания. Этот элемент представляет работников, которые имеют состояние **«занят»**, **об отсутствии на работе**или **под вопросом**.
