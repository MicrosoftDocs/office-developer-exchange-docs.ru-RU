# <a name="callstate-um-web-service"></a>Каллстате (веб-служба единой системы обмена сообщениями)

Элемент **каллстате** содержит значение, указывающее состояние вызова. 
  
[Жеткаллинфореспонсе (веб-служба единой системы обмена сообщениями)](getcallinforesponse-um-web-service.md)
  
[Каллстате (веб-служба единой системы обмена сообщениями)](callstate-um-web-service.md)
  
```xml
<CallState/>
```

 **умкаллстате**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.
  
### <a name="child-elements"></a>Дочерние элементы

Отсутствуют.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Жеткаллинфореспонсе (веб-служба единой системы обмена сообщениями)](getcallinforesponse-um-web-service.md) <br/> |Определяет ответ на [операцию GetCallInfo (веб-службу единой системы обмена сообщениями)](getcallinfo-operation-um-web-service.md).  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Необходимо указать текстовое значение. Ниже перечислены возможные значения.
  
- Простоя
    
- Connecting
    
- Предупрежден
    
- Подключено
    
- Отключено
    
- Входящее
    
- Мещают
    
- Сообщений
