# <a name="formattedaddress"></a>форматтедаддресс

Элемент **форматтедаддресс** указывает отформатированное отображаемое значение связанного почтового адреса. 
  
```XML
<FormattedAddress></FormattedAddress>
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.
  
### <a name="child-elements"></a>Дочерние элементы

Отсутствуют.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Значение (Персонапосталаддресстипе)](value-personapostaladdresstype.md) <br/> |Указывает сведения, связанные с почтовым адресом.  <br/> |
|[Посталаддресс (Персонапосталаддресстипе)](postaladdress-personapostaladdresstype.md) <br/> |Указывает почтовый адрес расположения.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **форматтедаддресс** — это строковое значение, задающее отформатированный адрес. 
  
## <a name="remarks"></a>Примечания

Этот элемент появился в Exchange Server 2013.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
