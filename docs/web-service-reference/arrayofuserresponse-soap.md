# <a name="arrayofuserresponse-soap"></a>ArrayOfUserResponse (SOAP)

Элемент **ArrayOfUserResponse** содержит массив элементов [Ответ пользователя (SOAP)](userresponse-soap.md) . 
  
```XML
<ArrayOfUserResponse>
   <UserResponse/>
</ArrayOfUserResponse>
```

 **ArrayOfUserResponse**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Ответ пользователя (SOAP)](userresponse-soap.md) <br/> |Содержит запрошенные параметры для указанного пользователя.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

Нет.
  
## <a name="text-value"></a>Текстовое значение
