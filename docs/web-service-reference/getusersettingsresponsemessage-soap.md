# <a name="getusersettingsresponsemessage-soap"></a>Жетусерсеттингсреспонсемессаже (SOAP)

Элемент **жетусерсеттингсреспонсемессаже** определяет ответ на запрос [операции GetUserSettings (SOAP)](getusersettings-operation-soap.md) . 
  
```XML
<GetUserSettingsResponseMessage>
   <Response/>
</GetUserSettingsResponseMessage>
```

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Отклик (SOAP)](response-soap.md) <br/> |Содержит ответ на запрос [операции GetUserSettings (SOAP)](getusersettings-operation-soap.md) .  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

Нет.
