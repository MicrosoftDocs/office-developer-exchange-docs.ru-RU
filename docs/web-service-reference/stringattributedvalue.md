# <a name="stringattributedvalue"></a>стрингаттрибутедвалуе

Элемент **стрингаттрибутедвалуе** указывает экземпляр в массиве атрибутов, связанных с элементом персоны. 
  
```XML
<StringAttributedValue>
   <Value/>
   <Attributions/>
</StringAttributedValue>
```
