# <a name="majorversion-soap"></a>MajorVersion (SOAP)

Элемент **MajorVersion** представляет основной номер версии для сервера. 
  
```XML
<MajorVersion/>
```

 **int**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[ServerVersionInfo (SOAP)](serverversioninfo-soap.md) <br/> |Содержит версию сервера, на обработку запроса.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение для элемента **MajorVersion** — целое число, представляющее основной номер версии сервера, на котором обработать запрос.
