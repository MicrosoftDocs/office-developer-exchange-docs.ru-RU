# <a name="actiontype-holdactiontype"></a>Себя (Холдактионтипе)

Элемент **тип** действия указывает тип действия для удержания. 
  
```XML
<ActionType> Create | Update | Remove </ActionType>
```

 **холдактионтипе**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

[SetHoldOnMailboxes](setholdonmailboxes.md)
  
## <a name="text-value"></a>Текстовое значение

Текстовое значение **элемента "объект** " — это тип удержания для почтового ящика. Текстовое значение **CREATE** указывает на то, что будет создано удержание почтового ящика. Текстовое значение **Update** указывает на то, что будет обновлено удержание почтового ящика. Текстовое значение **Remove** указывает на то, что удержание почтового ящика будет удалено. 
  
## <a name="remarks"></a>Примечания

Этот элемент появился в Exchange Server 2013.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
