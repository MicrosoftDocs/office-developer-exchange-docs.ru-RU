# <a name="originaldisplayname"></a>оригиналдисплайнаме

Элемент **оригиналдисплайнаме** содержит исходное отображаемое имя, связанное с адресом электронной почты. 
  
```XML
<OriginalDisplayName></OriginalDisplayName>
```

## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **оригиналдисплайнаме** — исходное отображаемое имя, связанное с адресом электронной почты. 
  
## <a name="remarks"></a>Примечания

Этот элемент появился в Exchange Server 2013.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
