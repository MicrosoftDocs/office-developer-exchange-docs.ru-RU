# <a name="unified-messaging-web-service-xml-elements-for-exchange"></a>Единой системы обмена сообщениями веб-службы XML элементов для Exchange

Найдите справочные сведения для XML-элементов, которые являются частью веб-службы единой системы обмена сообщениями (UM) в Exchange.
  
Веб-служба единой системы обмена сообщениями (UM) содержит множество элементов XML, которые используются для определения запросы и ответы, отправленные между клиентом и сервером. В этом разделе приведены сведения об отдельных элементов XML.
  
> [!NOTE]
>  Для версий Exchange, начиная с Exchange 2010, мы рекомендуем использовать операции единой системы обмена СООБЩЕНИЯМИ, доступные в [Веб-служб Exchange (EWS)](http://msdn.microsoft.com/library/60285497-0c4e-4e51-84e1-34dd6d89a5d8%28Office.15%29.aspx) вместо веб-службы единой системы обмена СООБЩЕНИЯМИ по следующим причинам: > функции единой системы обмена СООБЩЕНИЯМИ на основе веб-служб Exchange имеют первого класса Поддержка в управляемый API веб-служб Exchange. > В версиях Exchange, начиная с Exchange 2010 все новые возможности единой системы обмена СООБЩЕНИЯМИ будут добавлены к веб-служб Exchange, но не к веб-службе единой системы обмена СООБЩЕНИЯМИ.
