# <a name="schemaversionsupported"></a>счемаверсионсуппортед

Элемент **счемаверсионсуппортед** содержит версию схемы манифеста, поддерживаемую клиентом. 
  
```XML
<SchemaVersionSupported />
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.
  
### <a name="child-elements"></a>Дочерние элементы

Отсутствуют.
  
### <a name="parent-elements"></a>Родительские элементы

[GetAppManifests](getappmanifests.md)
  
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **счемаверсионсуппортед** содержит версию схемы манифеста, поддерживаемую клиентом. Это значение указывает, какие манифесты приложений должны быть возвращены клиенту в ответе. 
  
## <a name="remarks"></a>Примечания

Этот элемент появился в Exchange Server 2013 с пакетом обновления 1 (SP1).
