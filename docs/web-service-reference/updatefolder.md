# <a name="updatefolder"></a>UpdateFolder

Элемент **операцию UpdateFolder** представляет операцию, используемую для обновления свойств указанной папки. 
  
```xml
<UpdateFolder>
   <FolderChanges/>
</UpdateFolder>
```

 **упдатефолдертипе**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[фолдерчанжес](folderchanges.md) <br/> |Содержит коллекцию изменений для указанной папки.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

Нет.
  
## <a name="remarks"></a>Примечания

Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере с MicrosoftExchange Server 2007 и установленной ролью сервера клиентского доступа.
