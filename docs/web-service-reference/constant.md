# <a name="constant"></a>Константа

Элемент **константу** определяет значение константы в качестве ограничения. 
  
```xml
<Constant Value="" />
```

 **ConstantValueType**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

|**Атрибут**|**Описание**|
|:-----|:-----|
|**Значение** <br/> |Задает значение для сравнения в ограничении.  <br/> |
   
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Contains](contains.md) <br/> |Представляет выражение поиска, которое определяет, содержит ли данное свойство указанную константу строчного типа.  <br/> |
|[FieldURIOrConstant](fielduriorconstant.md) <br/> |Представляет свойство или постоянное значение для использования при сравнении с другого свойства.  <br/> |
   
## <a name="remarks"></a>Замечания

Строковое значение атрибута **Value** должен быть принудительного в тип, который вы пытаетесь сравнения. Например при сравнении соответствия константное значение свойства даты и времени строковое значение должно представлять даты и времени. 
  
Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере с MicrosoftExchange Server 2007 и установленной ролью сервера клиентского доступа.
