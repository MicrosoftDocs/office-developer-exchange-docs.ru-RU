# <a name="requesttype"></a>RequestType

Элемент **RequestType** определяет, является ли запрос на прокси-сервер между узлами или запросов между лесами. 
  
```xml
<RequestType>CrossSite or CrossForest</RequestType>
```

 **AvailabilityProxyRequestType**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

Этот элемент не имеет родительского в схеме. Данный элемент используется в заголовке SOAP. Дополнительные сведения об использовании этого элемента содержатся в файле WSDL.
  
## <a name="text-value"></a>Текстовое значение

Для этого элемента требуется указать текстовое значение. Ниже перечислены возможные значения.
  
- CrossSite
    
- CrossForest
