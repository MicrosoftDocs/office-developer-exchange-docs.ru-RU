# <a name="pox-autodiscover-xml-elements-for-exchange"></a>Элементы XML автоматического обнаружения POX для Exchange

Найдите XML элемент справочные сведения для веб-службы автообнаружения POX в Exchange.
  
В этом разделе документации основано на экземпляры элемент XML автоматического обнаружения POX, отправленные между клиентом и сервером. Служба автообнаружения POX не имеет опубликованную схему, можно обнаружить на сервере. В этом разделе приведены сведения об отдельных элементов XML автоматического обнаружения POX.
