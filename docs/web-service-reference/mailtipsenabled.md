# <a name="mailtipsenabled"></a>MailTipsEnabled

Элемент **MailTipsEnabled** указывает, доступна ли служба советы почты. 
  
```xml
<MailTipsEnabled>true | false</MailTipsEnabled>
```

 **Boolean**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[MailTipsConfiguration (MailTipsServiceConfiguration)](mailtipsconfiguration-mailtipsserviceconfiguration.md) <br/> |Содержит сведения о конфигурации службы для службы советы почты.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение этого элемента равно **true** , если доступна служба советы почты. Значение равно **false** , если служба советы почты недоступен. 
  
## <a name="remarks"></a>Замечания

Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
