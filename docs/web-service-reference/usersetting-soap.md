# <a name="usersetting-soap"></a>UserSetting (SOAP)

Элемент [UserSetting (SOAP)](usersetting-soap.md) представляет параметр с одним пользователем. 
  
```XML
<UserSetting>
    <Name />
</UserSetting>
```

 **UserSetting**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Имя (SOAP)](name-soap.md) <br/> |Представляет имя пользователя.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Параметры пользователя (SOAP)](usersettings-soap.md) <br/> |Представляет коллекцию параметров пользователя.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Нет.
