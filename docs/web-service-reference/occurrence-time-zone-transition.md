# <a name="occurrence-time-zone-transition"></a>Вхождение (переход часового пояса)

Элемент **вхождения** представляет число дней недели в месяце, в котором происходит переход часового пояса. 
  
```xml
<Occurrence/>
```

**int**

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.
  
### <a name="child-elements"></a>Дочерние элементы

Отсутствуют.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[рекуррингдайтранситион](recurringdaytransition.md) <br/> |Представляет переход часового пояса, который выполняется в один день каждого года.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение — это целое число, представляющее число дней недели в месяце, в котором происходит переход часового пояса. В приведенной ниже таблице перечислены возможные значения.
  
|**Значение**|**Описание**|
|:-----|:-----|
|1,1  <br/> |Первое вхождение указанного дня недели из начала месяца.  <br/> |
|2  <br/> |Второй экземпляр указанного дня недели из начала месяца.  <br/> |
|4  <br/> |Третий экземпляр указанного дня недели с начала месяца.  <br/> |
|4   <br/> |Четвертый экземпляр указанного дня недели с начала месяца.  <br/> |
|–1  <br/> |Первое вхождение указанного дня недели с конца месяца.  <br/> |
|–2  <br/> |Второй экземпляр указанного дня недели, начиная с конца месяца.  <br/> |
|–3  <br/> |Третий экземпляр указанного дня недели, начиная с конца месяца.  <br/> |
|— 4  <br/> |Четвертый экземпляр указанного дня недели, начиная с конца месяца.  <br/> |
   
## <a name="remarks"></a>Примечания

Схема, описывающая этот элемент, находится в виртуальном каталоге EWS компьютера, на котором работает сервер Microsoft Exchange с установленной ролью сервера клиентского доступа.
