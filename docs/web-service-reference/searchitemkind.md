# <a name="searchitemkind"></a>сеарчитемкинд

Элемент **сеарчитемкинд** указывает тип элементов, в которых выполняется поиск операции **финдмаилбоксстатистиксбикэйворд** . 
  
```XML
<SearchItemKind>Email | Meetings | Tasks | Notes | Docs | Journals | Contacts | Im | Voicemail | Faxes | Posts | Rssfeeds</SearchItemKind>
```

 **сеарчитемкиндтипе**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

[мессажетипес](messagetypes.md)
  
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **сеарчитемкинд** — это тип элемента, в котором выполняется поиск по ключевым словам. Следующий список содержит текстовые значения, которые можно использовать в элементе **сеарчитемкинд** . 
  
- **E-mail** — указывает на то, что в сообщениях электронной почты выполняется поиск по ключевым словам. 
    
- **Meetings** — указывает на то, что в собраниях выполняется поиск по ключевым словам. 
    
- **Tasks** — указывает на то, что в задачах выполняется поиск по ключевым словам. 
    
- **Примечания** . указывает на то, что поиск заметок выполняется для ключевых слов. 
    
- " **Документы** " — указывает на то, что в документах выполняется поиск по ключевым словам. 
    
- **Журналы** — указывает на то, что журналы просматриваются по ключевым словам. 
    
- **Contacts** — указывает, что в контактах выполняется поиск по ключевым словам. 
    
- **IM** . указывает на то, что поиск в мгновенных сообщениях осуществляется по ключевым словам. 
    
- **Голосовая** почта указывает на то, что поиск по голосовым сообщениям осуществляется по ключевым словам. 
    
- **Факсы** — указывает на то, что в факсах выполняется поиск по ключевым словам. 
    
- **Posts** — указывает на то, что в записи выполняется поиск по ключевым словам. 
    
- **Рссфидс** — указывает на то, что RSS-каналы ищут ключевые слова. 
    
## <a name="remarks"></a>Примечания

Этот элемент появился в Exchange Server 2013.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
