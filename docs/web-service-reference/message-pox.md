# <a name="message-pox"></a>Сообщение (POX)

Элемент **сообщение** содержит сообщение об ошибке Ошибка ответа службы автообнаружения. 
  
[Ошибка (POX)](error-pox.md)
  
[Сообщение (POX)](message-pox.md)
  
```xml
<Message/>
```

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Ошибка (POX)](error-pox.md) <br/> |Содержит возврату ошибки службы автообнаружения.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение представляет сообщение об ошибке для ошибки ответа службы автообнаружения.
