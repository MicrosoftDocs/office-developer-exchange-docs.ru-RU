# <a name="webclienturls-soap"></a>WebClientUrls (SOAP)

Элемент **WebClientUrls** представляет параметров пользователя, который содержит коллекцию элементов [WebClientUrl (SOAP)](webclienturl-soap.md) . 
  
[UserSetting (SOAP)](usersetting-soap.md)
  
[WebClientUrls (SOAP)](webclienturls-soap.md)
  
[WebClientUrl (SOAP)](webclienturl-soap.md)
  
```XML
<WebClientUrls>
     <WebClientUrl/>
</WebClientUrls>

```

 **WebClientUrls**

Нет.

|**Элемент**|**Описание**|
|:-----|:-----|
|[WebClientUrl (SOAP)](webclienturl-soap.md) <br/> |Представляет URL-адрес клиента Exchange.  <br/> |
