# <a name="smtpaddress-soap"></a>SmtpAddress (SOAP)

Элемент **SmtpAddress** представляет альтернативный SMTP-адрес почтового ящика. 
  
```XML
<SmtpAddress/>
```

**строка**

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.
  
### <a name="child-elements"></a>Дочерние элементы

Отсутствуют.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Алтернатемаилбокс (SOAP)](alternatemailbox-soap.md) <br/> |Представляет альтернативный почтовый ящик.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **SmtpAddress** — адрес электронной почты SMTP альтернативного почтового ящика.
