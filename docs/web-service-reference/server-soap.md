# <a name="server-soap"></a>Сервер (SOAP)

Элемент **Server** представляет сервер альтернативного почтового ящика. 
  
```XML
<Server/>
```

 **string**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[AlternateMailbox (SOAP)](alternatemailbox-soap.md) <br/> |Представляет альтернативного почтового ящика.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **сервера** — это имя сервера, альтернативного почтового ящика.
