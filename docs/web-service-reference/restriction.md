# <a name="restriction"></a>Ограничение

Элемент **restriction** представляет ограничение или запрос, используемый для фильтрации элементов или папок в операциях FindItem/FindFolder и папках поиска. 
  
```xml
<Restriction>
   <SearchExpression/>
</Restriction>
```

 **рестриктионтипе**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.

|**Элемент**|**Описание**|
|:-----|:-----|
|[And](and.md) <br/> |Представляет выражение поиска, которое позволяет выполнять логическое действие **и** операцию между двумя или более выражениями поиска.  <br/> |
|[Contains](contains.md) <br/> |Представляет выражение поиска, которое определяет, содержит ли данное свойство предоставленное строковое значение константы.  <br/> |

|[исгреатерсан](isgreaterthan.md) <br/> |Представляет выражение поиска, которое сравнивает свойство со значением константы или другим свойством и возвращает **значение true** , если первое свойство больше значения или свойства.  <br/> |
|[исгреатерсанорекуалто](isgreaterthanorequalto.md) <br/> |Представляет выражение поиска, которое сравнивает свойство со значением константы или другим свойством и возвращает **значение true** , если первое свойство больше или равно значению или свойству.  <br/> |
