# <a name="targetapplicationuri-soap"></a>TargetApplicationUri (SOAP)

Элемент **TargetApplicationUri** определяет конечное приложение URI. Элемент **TargetApplicationUri** — только для внутреннего использования. Этот элемент не используется с клиентами. 
  
```XML
<TargetApplicationUri/>
```

 **anyURI**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[OrganizationRelationshipSettings (SOAP)](organizationrelationshipsettings-soap.md) <br/> |Представляет список отношений организации для одной организации  <br/> |
   
## <a name="remarks"></a>Замечания

Этот элемент определяет целевой URI-адрес внешней организации.
