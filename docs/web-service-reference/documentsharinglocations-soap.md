# <a name="documentsharinglocations-soap"></a>Документшаринглокатионс (SOAP)

Элемент **документшаринглокатионс** содержит список расположения и сведения метаданных для расположения общего доступа к документу. 
  
```XML
<DocumentSharingLocations>
   <DocumentSharingLocation />
</DocumentSharingLocations>
```

 **документшаринглокатионс**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Документшаринглокатион (SOAP)](documentsharinglocation-soap.md) <br/> |Содержит расположение и метаданные для расположения общего доступа к документу.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Документшаринглокатионколлектионсеттинг (SOAP)](documentsharinglocationcollectionsetting-soap.md) <br/> |Представляет параметр пользователя, который представляет собой коллекцию местоположений и метаданных общего доступа к документации.  <br/> |
