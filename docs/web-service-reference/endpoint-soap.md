# <a name="endpoint-soap"></a>Конечная точка (SOAP)

Элемент **конечной точки** Указывает конечную точку службы маркеров безопасности. 
  
```XML
<Endpoint/>
```

 **xs: anyURI**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет
  
### <a name="child-elements"></a>Дочерние элементы

Нет
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[TokenIssuer (SOAP)](tokenissuer-soap.md) <br/> |Указывает URI и конечной точки для службы маркеров безопасности.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение представляет конечной точки для службы маркеров web безопасности.
  
## <a name="remarks"></a>Замечания

Конечная точка используется для обмена данными с помощью маркеров веб-службы безопасности.
