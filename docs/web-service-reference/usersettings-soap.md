# <a name="usersettings-soap"></a>Параметры пользователя (SOAP)

**Параметры пользователя** элемент представляет коллекцию параметров пользователя. 
  
```XML
<UserSettings>
    <UserSetting/>
</UserSettings>
```

 **Параметры пользователя**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[UserSetting (SOAP)](usersetting-soap.md) <br/> |Представляет параметр с одним пользователем.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Ответ пользователя (SOAP)](userresponse-soap.md) <br/> |Представляет ответ на запрос GetUserSettings для отдельного пользователя.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Нет.
