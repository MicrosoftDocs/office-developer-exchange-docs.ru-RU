# <a name="settelephoneaccessfolderemail-um-web-service"></a>SetTelephoneAccessFolderEmail (веб-служба единой системы обмена СООБЩЕНИЯМИ)

Элемент **SetTelephoneAccessFolderEmail** определяет запрос для установки папки электронной почты по умолчанию, из которого единой системы обмена сообщениями будет читать сообщения по телефону. 
  
[SetTelephoneAccessFolderEmail (веб-служба единой системы обмена СООБЩЕНИЯМИ)](settelephoneaccessfolderemail-um-web-service.md)
  
```xml
<SetTelephoneAccessFolderEmail>
  <base64FolderId>   </base64FolderId>
</SetTelephoneAccessFolderEmail>
```

 **complexType**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[base64FolderId (веб-служба единой системы обмена СООБЩЕНИЯМИ)](base64folderid-um-web-service.md) <br/> |Идентификатор папки электронной почты.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

Нет.
