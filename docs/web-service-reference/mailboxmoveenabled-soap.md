# <a name="mailboxmoveenabled-soap"></a>MailboxMoveEnabled (SOAP)

Элемент **MailBoxMoveEnabled** представляет флаг **MailboxMoveEnabled()** . Элемент **MailBoxMoveEnabled** — только для внутреннего использования. Этот элемент не используется с клиентами. 
  
```XML
<MailBoxMoveEnabled>true | false</MailBoxMoveEnabled>
```

**Boolean**

## <a name="attributes-and-elements"></a>Атрибуты и элементы
