# <a name="endpoint-soap"></a>Конечная точка (SOAP)

Элемент **Endpoint** указывает конечную точку службы маркеров безопасности. 
  
```XML
<Endpoint/>
```

 **xs: Анюри**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Нет
  
### <a name="child-elements"></a>Дочерние элементы

Нет
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Токениссуер (SOAP)](tokenissuer-soap.md) <br/> |Указывает URI и конечную точку для службы маркеров безопасности.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение представляет конечную точку для веб-службы маркеров безопасности.
  
## <a name="remarks"></a>Примечания

Конечная точка используется для связи с веб-службой маркеров безопасности.
