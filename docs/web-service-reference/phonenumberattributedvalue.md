# <a name="phonenumberattributedvalue"></a>фоненумбераттрибутедвалуе

Элемент **фоненумбераттрибутедвалуе** указывает экземпляр массива номеров телефонов и связанные с ними атрибуты. 
  
```XML
<PhoneNumberAttributedValue>
   <Value/>
   <Attributions/>
</PhoneNumberAttributedValue>
```
