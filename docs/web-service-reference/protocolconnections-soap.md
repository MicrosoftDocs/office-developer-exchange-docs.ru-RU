# <a name="protocolconnections-soap"></a>Протоколконнектионс (SOAP)

Элемент **протоколконнектионс** содержит ноль или более подключений протокола. 
  
```XML
<ProtocolConnections>
   <ProtocolConnection/>
</ProtocolConnections>
```

 **протоколконнектионс**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Протоколконнектионс (SOAP)](protocolconnections-soap.md) <br/> |Содержит ноль или более подключений протоколов.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Протоколконнектионколлектионсеттинг (SOAP)](protocolconnectioncollectionsetting-soap.md) <br/> |Представляет коллекцию параметров подключения по протоколу сервера.  <br/> |
   
## <a name="text-value"></a>Текстовое значение
