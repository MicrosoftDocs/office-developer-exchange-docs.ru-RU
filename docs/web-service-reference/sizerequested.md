# <a name="sizerequested"></a>сизерекуестед

Элемент **сизерекуестед** содержит запрошенный размер фотографии для операции **GetUserPhoto** . 
  
```XML
<SizeRequested>HR48x48 | HR64x64 | HR96X96 | HR120X120 | HR240X240 | HR360X360 | HR432X432 | HR504X504 | HR648X648</SizeRequested>
```

 **усерфотосизетипе**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.
  
### <a name="child-elements"></a>Дочерние элементы

Отсутствуют.
  
### <a name="parent-elements"></a>Родительские элементы

[GetUserPhoto](getuserphoto.md)
  
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **сизерекуестед** — это запрошенный размер фотографии цифрового изображения, возвращенного с сервера. В следующей таблице приведены текстовые значения для элемента **сизерекуестед** . 
  
|**Значение**|**Смысл**|
|:-----|:-----|
|HR48x48  <br/> |Изображение составляет 48 пикселей в высоту и 48 пикселей в ширину.  <br/> |
|HR64x64  <br/> |Изображение составляет 64 пикселей в высоту и 64 пикселей в ширину.  <br/> |
|HR96x96  <br/> |Изображение составляет 96 пикселей в высоту и 96 пикселей в ширину.  <br/> |
|HR120x120  <br/> |Изображение составляет 120 пикселей в высоту и 120 пикселей в ширину.  <br/> |
|HR240x240  <br/> |Изображение составляет 240 пикселей в высоту и 240 пикселей в ширину.  <br/> |
|HR360x360  <br/> |Изображение составляет 360 пикселей в высоту и 360 пикселей в ширину.  <br/> |
|HR432x432  <br/> |Изображение составляет 432 пикселей в высоту и 432 пикселей в ширину.  <br/> |
|HR504x504  <br/> |Изображение составляет 504 пикселей в высоту и 504 пикселей в ширину.  <br/> |
|HR648x648  <br/> |Изображение составляет 648 пикселей в высоту и 648 пикселей в ширину.  <br/> |
   
## <a name="remarks"></a>Примечания

Этот элемент появился в Exchange Server 2013.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
