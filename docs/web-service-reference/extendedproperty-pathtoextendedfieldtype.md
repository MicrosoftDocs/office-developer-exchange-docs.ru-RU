# <a name="extendedproperty-pathtoextendedfieldtype"></a>ExtendedProperty (Пастоекстендедфиелдтипе)

Элемент **ExtendedProperty** указывает расширенное свойство для единого хранилища контактов. 
  
```xml
<ExtendedProperty DistinguishedPropertySetId="" PropertySetId="" PropertyTag="" PropertyName="" PropertyId="" PropertyType="" FieldURI="">
</ExtendedProperty>
```

**пастоекстендедфиелдтипе**

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

|**Атрибут**|**Описание**|
|:-----|:-----|
|дистингуишедпропертисетид  <br/> |Указывает идентификатор набора отличительных свойств. Этот атрибут является необязательным.  <br/> |
|пропертисетид  <br/> |Указывает идентификатор набора свойств GUID. Этот атрибут является необязательным.  <br/> |
|пропертитаг  <br/> | Представляет тег свойства за исключением части типа.<br/><br/>Существует два варианта представления:  <br/><br/>— Шестнадцатеричный формат: 0x3fa4  <br/>— Десятичное число: 0-65535<br/><br/>  Этот атрибут является необязательным.  <br/> |
|PropertyName  <br/> |Строка, указывающая имя свойства. Этот атрибут является необязательным.  <br/> |
|пропертид  <br/> |Целое число, указывающее идентификатор свойства. Этот атрибут является необязательным.  <br/> |
|PropertyType  <br/> |Указывает тип свойства. Этот атрибут является обязательным.  <br/> |
|фиелдури  <br/> |Указывает универсальный код ресурса (URI) для поля. Этот атрибут является обязательным. Возможные значения приведены в разделе [фиелдури](fielduri.md) .  <br/> |

## <a name="remarks"></a>Примечания

Этот элемент появился в Exchange Server 2013.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
