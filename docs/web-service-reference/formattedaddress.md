# <a name="formattedaddress"></a>FormattedAddress

Элемент **FormattedAddress** указывает форматированный отображаемое значение связанный почтовый адрес. 
  
```XML
<FormattedAddress></FormattedAddress>
```

 **string**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Значение (PersonaPostalAddressType)](value-personapostaladdresstype.md) <br/> |Задает сведения, связанные с почтовый адрес.  <br/> |
|[PostalAddress (PersonaPostalAddressType)](postaladdress-personapostaladdresstype.md) <br/> |Указывает почтовый адрес расположения.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **FormattedAddress** — это строковое значение, указывающее, отформатированный адрес. 
  
## <a name="remarks"></a>Замечания

Этот элемент появился в Exchange Server 2013.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
