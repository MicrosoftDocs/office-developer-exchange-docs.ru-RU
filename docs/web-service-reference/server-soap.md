# <a name="server-soap"></a>Сервер (SOAP)

Элемент **Server** представляет альтернативный сервер почтовых ящиков. 
  
```XML
<Server/>
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Алтернатемаилбокс (SOAP)](alternatemailbox-soap.md) <br/> |Представляет альтернативный почтовый ящик.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **Server** — имя альтернативного сервера почтовых ящиков.
