# <a name="lastresponsetime"></a>ластреспонсетиме

Элемент **ластреспонсетиме** представляет дату и время последнего полученного ответа. 
  
```xml
<LastResponseTime/>
```

 **DateTime**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.
  
### <a name="child-elements"></a>Дочерние элементы

Отсутствуют.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Участник](attendee.md) <br/> |Представляет участников и ресурсы собрания.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Необходимо указать текстовое значение, представляющее дату и время последнего ответа.
  
## <a name="remarks"></a>Примечания

Схема, описывающая этот элемент, находится в виртуальном каталоге EWS компьютера, на котором работает Майкрософт Exchange 2007, на котором установлена роль сервера клиентского доступа.
