# <a name="path"></a>Путь

Элемент **path** является базовым типом схемы для всех идентификаторов свойств. Этот тип является абстрактным и никогда не происходит непосредственно в экземплярах документов. 
  
```xml
<Path/>
```

 **басепастоелементтипе**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.
  
### <a name="child-elements"></a>Дочерние элементы

Отсутствуют.
  
### <a name="parent-elements"></a>Родительские элементы

Нет.
  
## <a name="remarks"></a>Примечания

Для замены элемента **path** используются следующие элементы: 
  
- [фиелдури](fielduri.md)
    
- [индекседфиелдури](indexedfielduri.md)
    
- [ексцептионфиелдури](exceptionfielduri.md)
    
- [екстендедфиелдури](extendedfielduri.md)
    
Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере, работающем под управлением Microsoft Exchange Server 2007, с установленной ролью сервера клиентского доступа.
