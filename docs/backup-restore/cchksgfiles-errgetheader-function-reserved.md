# <a name="cchksgfileserrgetheader-function-reserved"></a>Функция CChkSGFiles.ErrGetHeader (зарезервировано)

**Применимо к:** Exchange Server 2003 | Exchange Server 2007 | Exchange Server 2010 | Exchange Server 2013
  
Зарезервировано для будущего использования, а не реализовано. Эта функция не вызывается. 
  
```cs
Vitual ERRErrGetHeader  
(
    __in_z const WCHAR  * const wszFile,
        VOID  * const pvResult,
    Const ULONGcbMax,
    Const ULONGulFlags = NO_FLAGS
);

```

## <a name="parameters"></a>Parameters

### <a name="wszfile"></a>wszFile
  
Зарезервировано для будущего использования; не следует использовать.
    
### <a name="pvresult"></a>pvResult
  
Зарезервировано для будущего использования; не следует использовать.
    
### <a name="cbmax"></a>cbMax
  
Зарезервировано для будущего использования; не следует использовать.
    
### <a name="ulflags"></a>ulFlags
  
Зарезервировано для будущего использования; не следует использовать.
    
## <a name="return-value"></a>Возвращаемое значение

Нет.
  
## <a name="remarks"></a>Замечания

Нет.
  
## <a name="requirements"></a>Требования

Exchange Server 2013 включает в себя только 64-разрядная версия CHKSGFILES API-интерфейса.
