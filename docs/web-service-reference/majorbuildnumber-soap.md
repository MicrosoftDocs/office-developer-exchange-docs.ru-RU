# <a name="majorbuildnumber-soap"></a>MajorBuildNumber (SOAP)

Элемент **MajorBuildNumber** представляет основной номер построения для сервера. 
  
```XML
<MajorBuildNumber/>
```

 **xs: int**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[ServerVersionInfo (SOAP)](serverversioninfo-soap.md) <br/> |Содержит версию сервера, на обработку запроса.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента MajorBuildNumber — целое число, представляющее основной номер сервера, на обработку запроса сборки.
