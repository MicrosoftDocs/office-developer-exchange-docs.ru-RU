# <a name="displayto"></a>DisplayTo

Элемент **DisplayTo** представляет строку отображения, который используется для содержимого в поле «Кому». Это составное строка всех для получателей отображаемые имена. 
  
```xml
<DisplayTo/>
```

 **Строка**

Текстовое значение представляет строку отображения, который используется для содержимого в поле «Кому».

Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере, работающем под управлением Microsoft Exchange Server 2007, с установленной ролью сервера клиентского доступа.
