# <a name="majorversion-soap"></a>MajorVersion (SOAP)

Элемент **majorversion** представляет основной номер версии сервера. 
  
```XML
<MajorVersion/>
```

 **int**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.
  
### <a name="child-elements"></a>Дочерние элементы

Отсутствуют.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Серверверсионинфо (SOAP)](serverversioninfo-soap.md) <br/> |Содержит версию сервера, который обработал запрос.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение для элемента **majorversion** — это целое число, представляющее основной номер версии сервера, который обработал запрос.
