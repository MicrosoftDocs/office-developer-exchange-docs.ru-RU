# <a name="appstatus"></a>аппстатус

Значение элемента **аппстатус** указывает состояние почтового приложения. 
  
```XML
<AppStatus/>
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

[Метаданные](metadata-ex15websvcsotherref.md)
  
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **аппстатус** указывает состояние почтового приложения. Если пользователь может устранить проблему, связанную с состоянием почтового приложения, элемент [актионурл](actionurl.md) предоставляет URL-адрес для выполнения исправления. 
  
**Таблица 1. Значения Аппстатус**

|**Значение**|**Описание**|
|:-----|:-----|
|NULL или 0  <br/> |Почтовое приложение имеет состояние работоспособности.  <br/> |
|1.0  <br/> |Почтовое приложение не удалось обновить автоматически. Почтовое приложение необходимо повторно установить из магазина Office.  <br/> |
|1.1  <br/> |Почтовое приложение не удалось обновить автоматически. Почтовому приложению требуются расширенные разрешения, и для этого требуется проверка и подтверждение установки.  <br/> |
|1.2  <br/> |Не удалось автоматически обновить почтовое приложение. Текущая лицензия просрочена или недействительна. Обновите почтовое приложение из магазина Office.  <br/> |
|2.0  <br/> |Не удалось автоматически обновить лицензию почтового приложения. Лицензия для почтового приложения должна быть восстановлена из магазина Office.  <br/> |
|2.1  <br/> |Не удалось автоматически обновить лицензию почтового приложения. Срок действия текущей лицензии истек. Новая лицензия для этого приложения должна быть установлена из магазина Office.  <br/> |
|3,0  <br/> |Состояние магазина Office для почтового приложения изменилось. Это может означать, что в почтовом приложении возникла проблема. Чтобы получить дополнительные сведения, перейдите на страницу почтового приложения в магазине Office.  <br/> |
|3.1  <br/> |Почтовое приложение удалено из магазина Office.  <br/> |
|3.2  <br/> |В почтовом приложении обнаружена проблема, которая временно снята из магазина Office.  <br/> |
|3.3  <br/> |Почтовое приложение будет удалено из магазина Office в течение 30 дней.  <br/> |
|4,0  <br/> |Почтовый клиент автоматически отключил почтовое приложение.  <br/> |
|4.1  <br/> |Почтовые приложения отключены в Outlook по соображениям производительности.  <br/> |
   
## <a name="remarks"></a>Примечания

Этот элемент появился в Exchange Server 2013 с пакетом обновления 1 (SP1).
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
