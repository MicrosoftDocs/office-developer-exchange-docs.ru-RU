# <a name="fileextension-soap"></a>FileExtension (SOAP)

Элемент **fileExtension** представляет расширение файла. 
  
```XML
<FileExtension/>
```

 **строка**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Суппортедфиликстенсионс (SOAP)](supportedfileextensions-soap.md) <br/> |Содержит список расширений файлов в расположении общего доступа к документу.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **fileExtension** является допустимым расширением файла для расположения общего доступа к документу. 
  
## <a name="element-information"></a>Сведения об элементе

|||
|:-----|:-----|
|Пространство имен  <br/> |http://schemas.microsoft.com/exchange/2010/Autodiscover  <br/> |
|Имя схемы  <br/> |Схема автообнаружения  <br/> |
|Файл проверки  <br/> |Messages. xsd  <br/> |
|Может быть пустым  <br/> |True  <br/> |
