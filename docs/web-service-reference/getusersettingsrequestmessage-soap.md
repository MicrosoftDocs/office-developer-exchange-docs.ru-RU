# <a name="getusersettingsrequestmessage-soap"></a>Жетусерсеттингсрекуестмессаже (SOAP)

Элемент **жетусерсеттингсрекуестмессаже** представляет запрос [операции GetUserSettings (SOAP)](getusersettings-operation-soap.md) . 
  
```XML

</GetUserSettingsRequestMessage>
```
