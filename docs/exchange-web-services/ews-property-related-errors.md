# <a name="ews-property-related-errors"></a>Ошибки, связанные со свойством EWS

Узнайте, как обрабатывать ошибки, связанные с свойствами, в приложении EWS.
  
Большинство клиентских приложений EWS будут использовать свойства, что означает, что вам придется обрабатывать ошибки, связанные со свойствами. Эти ошибки можно обрабатывать во время выполнения или при разработке приложения EWS.
  
**Таблица 1: ошибки, связанные со свойствами, и способы их обработки**

|**Error**|**Вызывается при попытке...**|**Обработать его по...**|
|:-----|:-----|:-----|
|еррордатасизелимитексцеедед  <br/> |Задайте для свойства значение, превышающее максимальный размер свойства, или свойство не поддерживает потоковую передачу, такую как свойства папки.  <br/> |Ограничение размера данных, устанавливаемых для свойства.  <br/> |
|еррорфолдерпропертрекуестфаилед  <br/> |Получение свойства, которое не удалось получить.  <br/> |Указывает, что свойство не может быть извлечено.  <br/> |
|ерроринвалидекстендедпроперти  <br/> |Задание недопустимого сочетания значений расширенных свойств или приводит к неправильному универсальному идентификатору ресурса (URI) расширенного свойства.  <br/> |Проверка значения расширенного свойства.  <br/> |
|ерроринвалидекстендедпропертивалуе  <br/> |Задание значения расширенного свойства, которое не отвечает заданному типу  <br/> |Обновление кода для проверки совпадений типов.  <br/> |
|ерроринвалидфолдерид  <br/> |Присвойте структуре идентификатора папки недопустимую форму.  <br/> |Только с помощью идентификаторов, возвращенных службой EWS.  <br/> |
|ерроринвалидид  <br/> |Установите структуру идентификатора и/или измените ключ на недопустимую форму.  <br/> |Только с помощью идентификаторов, возвращенных службой EWS.  <br/> |
|ерроринвалидидемпти  <br/> |Задайте пустой идентификатор.  <br/> |Задание идентификатора с допустимым идентификатором элемента или папки.  <br/> |
|ерроринвалидидмалформед  <br/> |Установите структуру идентификатора и/или измените ключ на недопустимую форму.  <br/> |Только с помощью идентификаторов, возвращенных службой EWS.  <br/> |
|ерроринвалидпропертяппенд  <br/> |Добавьте свойство, которое не поддерживает добавление.  <br/> |Обновление кода таким образом, чтобы он пытался только добавить значения в свойства коллекции получателей (to, CC, BCC), свойства коллекции участников (обязательные, необязательные, ресурсы), свойство Body и свойство ReplyTo.  <br/> |
|ерроринвалидпропертиделете  <br/> |Удаление свойства, которое не поддерживает удаление.  <br/> |Обновление кода без попытки удалить свойство. Например, невозможно удалить идентификаторы папок и элементов.  <br/> |
|ерроринвалидпропертифорексистс  <br/> |Задайте ограничение поиска на основе ексистентиал для свойства на основе флагов.  <br/> |Обновление кода без использования свойств на основе флагов в ограничении поиска на основе ексистентиал. Для свойств, основанных на флагах, применяются свойства исресенд, Исунмодифиед, и Исфромме.  <br/> |
|ерроринвалидпропертифороператион  <br/> |Работа с свойством элемента или папки, которые не поддерживаются операцией.  <br/> |Обновление кода для того, чтобы не получить доступ к свойству с операцией, вызвавшей ошибку.  <br/> |
|ерроринвалидпропертирекуест  <br/> |Укажите свойство в запросе, которое не поддерживается для типа элемента.  <br/> |Обновление кода, чтобы не пытаться получить доступ к свойству с помощью операции.  <br/> |
|ерроринвалидпропертисет  <br/> |Задайте свойство, доступное только для чтения.  <br/> |Обновление кода для того, чтобы не пытаться задать свойство.  <br/> |
|ерроринвалидвалуефорпроперти  <br/> |Сравните значение свойства в ограничении поиска, в котором значение сравнения не отвечает типу свойства.  <br/> |Обновление кода для проверки несовпадения типов свойств.  <br/> |
|ерроритемсавепропертеррор  <br/> |Сохранение элемента или папки с недопустимыми значениями свойств.  <br/> |Проверка значений и типов свойств перед их отправкой в запросе.  <br/> |
|еррорнофолдерклассоверриде  <br/> |Задайте класс Folder для новой папки, которая не является базовым типом папки.  <br/> |Использование универсального типа папки для задания класса папки.  <br/> |
|еррорнопропертитагфоркустомпропертиес  <br/> |Обращение к настраиваемому расширенному свойству с помощью тега Property.  <br/> |Обновление кода для ссылки на настраиваемое расширенное свойство по идентификатору набора свойств и по имени свойства или по идентификатору диспетчеризации свойства.  <br/> |
|ерроробжекттипечанжед  <br/> |Задайте или обновите класс элемента для элемента, который не сопоставлен с типом схемы.  <br/> |Обновление кода таким образом, чтобы класс элемента соответствовал типу схемы элемента.  <br/> |
|еррорпропертюпдате  <br/> |Обновление свойства с недопустимым значением свойства.  <br/> |Проверка значения свойства перед его отправкой в запросе [UpdateItem](https://msdn.microsoft.com/library/5d027523-e0bc-4da2-b60b-0cb9fc1fdfe4%28Office.15%29.aspx) .  <br/> |
|ерроррекуиредпропертимиссинг  <br/> |Отправить запрос CreateAttachment, в котором отсутствует обязательное свойство.  <br/> |Обновление кода для установки отсутствующего свойства в соответствии с указанием пути свойства, возвращаемого в ответе.  <br/> |
|еррорунсуппортедмапипропертитипе  <br/> |Используйте расширенные типы свойств Type Object, Object Array, Error или null.  <br/> |Обновление кода без использования ограниченных расширенных типов свойств.  <br/> |

|еррорупдатепропертимисматч  <br/> |Обновление элемента или папки. Описание изменения не отвечает свойству, которое было указано для обновления.  <br/> |Изменение кода таким образом, чтобы описание изменения соответствовало обновляемому типу элемента или папки.  <br/> |
