<span data-ttu-id="aa30f-163">В следующем примере показан SOAP-запрос SOAP для получения параметров домена из службы автообнаружения.</span><span class="sxs-lookup"><span data-stu-id="aa30f-163">The following example shows a SOAP XML request to get both domain settings from the Autodiscover service.</span></span>

```XML
<?xml version="1.0" encoding="utf-8"?>

        </a:Domains>
        <a:RequestedSettings>
          <a:Setting>ExternalEwsUrl</a:Setting>
          <a:Setting>ExternalEwsVersion</a:Setting>
        </a:RequestedSettings>
        <a:RequestedVersion>Exchange2013</a:RequestedVersion>
      </a:Request>
    </a:GetDomainSettingsRequestMessage>
  </soap:Body>
</soap:Envelope>
```
