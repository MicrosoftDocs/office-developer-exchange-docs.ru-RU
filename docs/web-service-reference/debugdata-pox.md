# <a name="debugdata-pox"></a>DebugData (POX)

Элемент **DebugData** содержит данные отладки для возврату ошибки службы автообнаружения. 
  
- [Ошибка (POX)](error-pox.md) 
- [DebugData (POX)](debugdata-pox.md)
  
```xml
<DebugData/>
```

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Ошибка (POX)](error-pox.md) <br/> |Содержит возврату ошибки службы автообнаружения.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение представляет данные отладки для возврату ошибки службы автообнаружения.
