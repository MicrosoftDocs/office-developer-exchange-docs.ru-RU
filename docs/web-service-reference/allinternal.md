# <a name="allinternal"></a>AllInternal

Элемент **AllInternal** имеет значение **true** , если все получатели сообщения электронной почты являются внутренними относительно организации отправителя. 
  
```xml
<AllInternal/>
```

 **ProtectionRuleAllInternalType**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Условие](condition.md) <br/> |Определяет условие, которое должно выполняться для части действия правила для выполнения.  <br/> |
|[И (ProtectionRuleAndType)](and-protectionruleandtype.md) <br/> |Указывает, что все дочерние элементы должны соответствовать значение **true**.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Элемент **AllInternal** должен быть пустым. 
  
## <a name="remarks"></a>Замечания

Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
