# <a name="absolutedate"></a>AbsoluteDate

Элемент **AbsoluteDate** представляет дату, когда изменяется время от стандартное или летнее время. 
  
```xml
<AbsoluteDate/>
```

**Дата**

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Стандартный](standard.md) <br/> |Представляет дату и время изменения времени с летнего времени на стандартное время.  <br/> |
|[Переход на летнее](daylight.md) <br/> |Представляет дату и время изменения времени с зимнего на летнее время.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение представляет дату, когда происходит shift между стандартное или летнее время.
  
## <a name="remarks"></a>Замечания

Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере с MicrosoftExchange Server 2007 и установленной ролью сервера клиентского доступа.
