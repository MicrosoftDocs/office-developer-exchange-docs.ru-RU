# <a name="schemaversionsupported"></a>SchemaVersionSupported

Элемент **SchemaVersionSupported** содержит версию схемы манифеста, который поддерживается клиентом. 
  
```XML
<SchemaVersionSupported />
```

 **string**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

[GetAppManifests](getappmanifests.md)
  
## <a name="text-value"></a>Текстовое значение

Текстовое значение элемента **SchemaVersionSupported** содержит версию схемы манифеста, который поддерживается клиентом. Это значение указывает, какое приложение манифесты должны быть возвращены клиенту в ответе. 
  
## <a name="remarks"></a>Замечания

Этот элемент появился в Exchange Server 2013 с пакетом обновления 1 (SP1).
