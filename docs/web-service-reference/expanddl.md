# <a name="expanddl"></a>ExpandDL

Элемент **ExpandDL** определяет запрос на расширение списка рассылки. 
  
```xml
<ExpandDL>
   <Mailbox/>
</ExpandDL>
```

 **експанддлтипе**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Нет
  
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[Mailbox](mailbox.md) <br/> |Определяет полностью разрешенный адрес электронной почты или список рассылки. Этот почтовый ящик представляет список рассылки, который требуется развернуть.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

Нет.
  
## <a name="remarks"></a>Примечания

Расширение списка рассылки будет выполняться только для одного списка рассылки. Расширение списка рассылки не является рекурсивным.
  
Схема, описывающая этот элемент, расположена в виртуальном каталоге EWS на компьютере с MicrosoftExchange Server 2007 и установленной ролью сервера клиентского доступа.
