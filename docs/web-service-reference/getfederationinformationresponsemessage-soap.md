# <a name="getfederationinformationresponsemessage-soap"></a>Жетфедератионинформатионреспонсемессаже (SOAP)

Элемент **жетфедератионинформатионреспонсемессаже** определяет ответ на запрос [операции жетфедератионинформатион (SOAP)](getfederationinformation-operation-soap.md) . 
  
```XML
<GetFederationInformationResponseMessage>
   <Response/>
</GetFederationInformationResponseMessage>
```

 **жетфедератионинформатионреспонсемессаже**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

Нет.
