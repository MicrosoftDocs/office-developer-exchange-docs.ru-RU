# <a name="mailboxmoveenabled-soap"></a>Маилбоксмовинаблед (SOAP)

Элемент **маилбоксмовинаблед** представляет флаг **маилбоксмовинаблед ()** . Элемент **маилбоксмовинаблед** предназначен только для внутреннего использования. Этот элемент не используется клиентами. 
  
```XML
<MailBoxMoveEnabled>true | false</MailBoxMoveEnabled>
```

**Boolean**
