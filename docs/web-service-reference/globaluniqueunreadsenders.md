# <a name="globaluniqueunreadsenders"></a>GlobalUniqueUnreadSenders

Элемент **GlobalUniqueUnreadSenders** Указывает список всех людей, отправленные сообщения, в настоящее время непрочитанные сообщения в беседе для всех папок в почтовом ящике.
