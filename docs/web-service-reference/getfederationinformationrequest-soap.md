# <a name="getfederationinformationrequest-soap"></a>GetFederationInformationRequest (SOAP)

Элемент **GetFederationInformationRequest** представляет параметры звонка на операции [GetFederationInformation операции (SOAP)](getfederationinformation-operation-soap.md).
  
```XML
<GetFederationInformationRequest>
   <Domain/>
</GetFederationInformationRequest>
```

**GetFederationInformationRequest**

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.

Нет.
