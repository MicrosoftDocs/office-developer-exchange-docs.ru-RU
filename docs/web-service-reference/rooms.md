# <a name="rooms"></a>Комнат

Элемент **комнат** — список из одного или нескольких элементов, которые представляют комнаты для собраний. 
  
[GetRoomsResponse](getroomsresponse.md)
  
[Комнат](rooms.md)
  
```xml
<Rooms>   <Room/></Rooms>
```

Нет.
