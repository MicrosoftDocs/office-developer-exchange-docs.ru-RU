# <a name="clientextension"></a>клиентекстенсион

Элемент **клиентекстенсион** содержит сведения о пользователе и конфигурации приложения. 
  
```XML
<ClientExtension IsAvailable=" true | false " IsMandatory=" true | false " IsEnabledByDefault=" true | false " Type="" Scope="" MarketplaceAssetId="" MarketplaceContentMarket="" AppStatus="" Etoken="">
    <SpecificUsers></SpecificUsers>
    <Manifest></Manifest>
</ClientExtension>
```

 **клиентекстенсионтипе**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

|**Атрибут**|**Описание**|
|:-----|:-----|
|Доступный доступ  <br/> |Указывает, доступно ли приложение. Текстовое значение **true** для атрибута **Available** указывает на то, что приложение доступно. Значение **false** указывает, что приложение недоступно. Этот атрибут является необязательным.  <br/> |
|Обязательный  <br/> |Указывает, является ли приложение обязательным. Текстовое значение **true** **для атрибута InAttribute** указывает на то, что приложение является обязательным для почтового ящика. Значение **false** указывает, что приложение не является обязательным. Этот атрибут является необязательным.  <br/> |
|исенабледбидефаулт  <br/> |Указывает, включено ли приложение по умолчанию. Текстовое значение **true** для атрибута **исенабледбидефаулт** указывает, что приложение включено по умолчанию. Значение **false** указывает, что приложение по умолчанию отключено. Этот атрибут является необязательным.  <br/> |
|ProvidedTo  <br/> |Указывает, кому предоставляется приложение. Этот атрибут является необязательным.  <br/> |
|Тип  <br/> |Указывает тип приложения.  <br/> |
|Scope  <br/> |Указывает область приложения.  <br/> |
|маркетплацеассетид  <br/> |Указывает идентификатор ресурса Marketplace для приложения.  <br/> |
|маркетплацеконтентмаркет  <br/> |Указывает контент Marketplace, который видит пользователь для получения сведений и рецензирования приложения.  <br/> |
|аппстатус  <br/> |Указывает код состояния почтового приложения в неожиданном состоянии.  <br/> |
|етокен  <br/> |Указывает маркер лицензии для почтовых или пробных почтовых приложений.  <br/> |
   
#### <a name="type"></a>Тип

|**Значение**|**Описание**|
|:-----|:-----|
|По умолчанию  <br/> |Указывает, что приложение доступно по умолчанию.  <br/> |
|Личные сведения  <br/> |Указывает, что приложение является частным.  <br/> |
|Рынках  <br/> |Указывает, что приложение является приложением Marketplace.  <br/> |
   
#### <a name="scope"></a>Scope

|**Значение**|**Описание**|
|:-----|:-----|
|Нет  <br/> |Указывает, что в приложении нет области.  <br/> |
|User  <br/> |Указывает, что приложение предназначено для каждого пользователя.  <br/> |
|Организация  <br/> |Указывает, что приложение предназначено для Организации.  <br/> |
|По умолчанию  <br/> |Указывает, что приложение является приложением по умолчанию.  <br/> |
   
### <a name="child-elements"></a>Дочерние элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[спеЦификусерс](specificusers.md) <br/> |Указывает учетные записи электронной почты, которые могут получать доступ к приложению.  <br/> |
|[Манифест](manifest.md) <br/> |Содержит файл манифеста приложения с кодировкой Base 64.  <br/> |
   
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[клиентекстенсионс](clientextensions.md) <br/> |Указывает массив элементов **клиентекстенсион** .  <br/> |
   
## <a name="remarks"></a>Примечания

Этот элемент появился в Exchange Server 2013.
  
Схема, описывающая этот элемент, находится в виртуальном каталоге IIS, в котором размещены веб-службы Exchange.
