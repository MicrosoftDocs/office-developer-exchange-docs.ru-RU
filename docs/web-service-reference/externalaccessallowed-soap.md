# <a name="externalaccessallowed-soap"></a>ExternalAccessAllowed (SOAP)

Элемент **ExternalAccessAllowed** указывает, является ли документ расположение для обмена доступны за пределами подключений. 
  
```XML
<ExternalAccessAllowed /> 
```

 **Boolean**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[DocumentSharingLocation (SOAP)](documentsharinglocation-soap.md) <br/> |Представляет сведения о расположении и метаданных для документа расположение для обмена.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Логическое значение элемента **ExternalAccessAllowed** указывает ли доступны за пределами подключения к месту общего доступа.
