# <a name="requesttype"></a>RequestType

Элемент **RequestType** определяет, является ли прокси-запрос межсайтым или запросом между лесами. 
  
```xml
<RequestType>CrossSite or CrossForest</RequestType>
```

 **аваилабилитипроксирекуесттипе**
## <a name="attributes-and-elements"></a>Атрибуты и элементы

В следующих разделах описываются атрибуты, дочерние и родительские элементы.
  
### <a name="attributes"></a>Атрибуты

Отсутствуют.
  
### <a name="child-elements"></a>Дочерние элементы

Отсутствуют.
  
### <a name="parent-elements"></a>Родительские элементы

Этот элемент не имеет родителя в схеме. Этот элемент используется в заголовке SOAP. Для получения дополнительных сведений об использовании этого элемента см файл WSDL.
  
## <a name="text-value"></a>Текстовое значение

Для этого элемента требуется указать текстовое значение. Ниже перечислены возможные значения.
  
- кросссите
    
- кроссфорест
