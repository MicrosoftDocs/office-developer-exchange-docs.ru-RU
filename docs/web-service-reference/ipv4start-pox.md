# <a name="ipv4start-pox"></a>IPv4Start (POX)

Элемент **IPv4Start** определяет начало диапазона IP версии 4 (IPv4) адреса, которые используются для идентификации компьютера в сети.

[Протокол (POX)](protocol-pox.md)
  
[NetworkRequirements (POX)](networkrequirements-pox.md)
  
[IPv4Start (POX)](ipv4start-pox.md)
  
```xml
<IPv4Start/>
```

## <a name="attributes-and-elements"></a>Атрибуты и элементы

В разделах ниже приводится описание атрибутов, дочерних и родительских элементов.
  
### <a name="attributes"></a>Атрибуты

Нет.
  
### <a name="child-elements"></a>Дочерние элементы

Нет.
  
### <a name="parent-elements"></a>Родительские элементы

|**Элемент**|**Описание**|
|:-----|:-----|
|[NetworkRequirements (POX)](networkrequirements-pox.md) <br/> |Содержит критерии, используемые для определения, является ли на клиентском компьютере в сети, с использованием Internet поставщика услуг (поставщика услуг Интернета) требования для подключения к серверу.  <br/> |
   
## <a name="text-value"></a>Текстовое значение

Текстовое значение представляет IPv4-адрес.
